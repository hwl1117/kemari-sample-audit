What I look for before suggesting a redesign.

Three UI opportunities

Small changes that can make the website feel more premium.

These are sample observations for an interior / fit-out website. A real client audit would reference the exact live pages and screenshots.

1

Make the first screen sell the level of work.

The first section should immediately show the best project image, the exact service category, and a clear consultation action.

2

Turn projects into proof, not just a gallery.

Project cards should show sector, size, location, scope, and result so a visitor can quickly find a relevant reference.

3

Make mobile inquiry feel intentional.

On mobile, the CTA should not be hidden. A simple fixed inquiry or WhatsApp entry can make the site easier to act on.
